A ultimate plugin to colorise the Mobile Chrome address bar and enable full mode and give style to iOS status bar.

== Description ==
Color Your Bar gives a simple way to change the color of address bar of Chrome browser for mobile devices. Color address bar works only on mobile version of Google Chrome. So you need to check changes on mobile device.

iOS support full mode and some basic status bar styles. Now you can allow to open your website in full mode with status bar style on iOS.

Use Color Your Bar is quite simple. Just install the plugin and enable it and choose your required color and save. That's simple.

----------NOW USE DIFFERNT COLOR ON EACH PAGE----------
Now you can color each post bar color with plugin's meta box.


== Installation ==
1. Search Color Your Bar in plugin menu on your WordPress site.
2. Download and activate the plugin.
3. Go to Color Your Bar
4. Check to enable it and chosse you required color from color picker. Save it.

== Frequently Asked Questions ==
= 1) How to Use? =
Navigate to Color Your Bar and enable it. Choose post types if you want to use different color bar for all your individual post. Choose your required color from color picker and save it.

For showing different color on post/page, edit post and choose color from color picker and update it.

= 2) Which browsers support it? =
Currently only Google Chrome mobile version support color. iOS support full screen mode and some status bar style.
